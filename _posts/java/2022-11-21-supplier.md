### Supplier

```java
public interface Supplier<T> {
    T get();
}
```

```java
//Static method Reference
Supplier<LocalDate> s1 = LocalDate::now;
//Lambda Expression
Supplier<LocalDate> s2 = () -> LocalDate.now();
```
